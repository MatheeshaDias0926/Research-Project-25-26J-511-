🚨 Crash Detection & Alert Flow

▼
Node.js Backend creates crash record & alerts
▼
SMS sent to Police Stations, Hospitals via Twilio
▼
Real-time crash banner appears on all dashboards

ESP32 counts passengers via IR sensors
▼
POST /api/iot/iot-data (every 5 min)
▼
BusDataLog stored in MongoDB
▼
Violation detection (footboard, overcrowding)
▼
Dashboard updates + SMS warnings
📐 Physics Safety Prediction Flow
User requests safety check (speed, load, GPS)
▼
Backend spawns Python physics subprocess
▼
OSMnx fetches real road curve ahead
▼
Physics engine calculates rollover risk & stopping distance
▼
Returns decision: Safe / Warning / Critical
